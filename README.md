# 🌍 Uyo Eco-Track: AI-Powered Waste-to-Wealth System

**Uyo Eco-Track** is a smart urban management solution designed to tackle the waste crisis and resulting flash floods in Uyo, Akwa Ibom State. This project bridges the gap between **Environmental Sustainability** and **Financial Inclusion** by transforming urban waste into a digital asset.

## 📍 The "Hidden Crisis" Circuit

While Uyo's main boulevards are beautifully maintained, critical "hidden" hotspots face severe indiscriminate dumping. This project maps the contrast between the clean "Surface" and the actual "Accumulation Zones":

* **Obio Imo Street (by Stadium):** A primary hotspot where waste accumulation threatens local schools and residents.
* **Uyo Township Stadium Entrance:** A high-traffic zone prone to illegal dumping despite its central location.
* **Udua Akpandem (Internal):** The economic heart of Uyo and a massive generator of PET plastic waste.
* **Aka Road & Nwaniba Arteries:** Logistics routes where even small amounts of waste lead to significant drain clogging and flash floods.

---

## 🚀 Key Features

### 💰 1. Waste-to-Wealth (Eco-Wallet)
To drive mass adoption in high-density areas like Udua Akpandem, we’ve built a circular economy loop:
* **Plastic Buy-Back:** Residents and traders can "sell" PET bottles directly through the app.
* **Digital Eco-Wallet:** Instant credit of points/cash upon verified collection at designated hubs.
* **Redeemable Value:** Points can be converted to airtime or bank transfers, incentivizing a "gutter-to-gold" mindset.

### 👁️ 2. AI Waste Validator
Using **Google Cloud Vision AI**, the system automatically verifies waste reports:
* Users snap a photo; the AI confirms it is valid waste (plastic, heap, or blocked drain).
* This filters out false reports and ensures collection teams prioritize high-impact areas like Obio Imo.

### 🗺️ 3. Dynamic Heatmap & Routing
* **Visual Hotspots:** Real-time mapping of waste density using Google maps infrastructure.
* **Logistics Optimization:** Calculates the most fuel-efficient route for trucks to move from the Plaza through Aka Road, targeting the specific hotspots.

This project was engineered entirely using mobile and cloud-based environments, proving that world-class urban solutions can be built with accessible hardware.

* **Frontend:** React (Vite)
* **Database & Hosting:** Firebase Firestore 
* **Cloud Infrastructure:** Google Cloud Platform 
* **APIs:** Cloud Vision AI, Google Maps JavaScript API, Geolocation API

## 📈 Impact & Scalability

* **Health & Safety:** Eliminating heaps at Obio Imo to protect school-aged children from environmental hazards.
* **Economic Empowerment:** Providing a secondary income stream for Uyo residents through plastic recycling.
* **Scalability:** A modular system architecture ready for deployment across all 31 LGAs in Akwa Ibom State.
